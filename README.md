# md001 juniper
Juniper Eurorack Module.

## firmware update instructions
Firmware may be updated by downloading one of the release binaries found in this repository. Firmware is upgraded by copying the a release binary to the FAT32 formatted microsd card of the device and naming the file `FW.BIN`. On powerup hold the `PHASE/CH7` button, the bootloader of the device will then flash the device with the desired firmware and the device will restart.

## release information
Currently the firmware is in closed early beta with various known bugs and some features not fully implemented.

## bug reporting
For any issues you find with the device, please create a new issue within this repository. Please ensure to list the device `revision` and `firmware`
